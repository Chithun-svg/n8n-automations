# n8n-automations
My 1st n8n AI automation workflows

A collection of AI-powered automation workflows built with n8n.

## 📋 Workflows

### 1. AI Email Summarizer
Automatically summarizes incoming Gmail emails using Google Gemini AI and sends the summary back to your inbox.

**Nodes used:**
- Gmail Trigger — watches for new emails
- Google Gemini — summarizes email into bullet points
- Gmail Send — delivers the summary to your inbox

**Tools needed:**
- n8n Cloud account
- Gmail (OAuth)
- Google Gemini API key (free at aistudio.google.com)

---

## 🚀 How to use these workflows

1. Download the `.json` file of the workflow you want
2. Open n8n → New Workflow → click `...` → Import
3. Add your own credentials (Gmail, API keys)
4. Activate and enjoy!
